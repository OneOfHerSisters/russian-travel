# Проект: Путешествие по России

Учебный проект по адаптивной вёрстке.  
Типовой одностраничный сайт.  
Содержит информацию об интересный местах России, куда можно съездить в путешествие.

## Выполнен с использованием:

1. Grid-layout
2. Флексбокс-верстка
3. Медиазапросы
4. Методология БЭМ.
